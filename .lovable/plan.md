

# בדיקת זרימה + מצב למידה בלבד + שיפורי הוראה

## 1. בדיקת הזרימה הנוכחית (תיאוריה → חימום → שאלה)
- הזרימה כבר מיושמת ב-`TopicPractice.tsx` עם `theoryDone` ו-`warmupDone` states
- `getTheoryForQuestion` מחזיר תיאוריה לכל שאלה (ספציפית או ברירת מחדל לפי נושא)
- צריך לוודא שאין באגים בפועל — אבדוק דרך הדפדפן אחרי היישום

## 2. הוספת מצב "למידה בלבד" (`/topic/:topicId/learn`)

| קובץ | שינוי |
|------|-------|
| `src/App.tsx` | Route חדש: `/topic/:topicId/learn` |
| `src/pages/TopicLearn.tsx` | דף חדש — מציג תיאוריה + חימום בלבד, ללא שאלה מלאה. בסיום מראה סיכום וכפתור "עכשיו תרגל!" |
| `src/pages/Topics.tsx` | הוספת כפתור "📖 למידה" ליד כל נושא (או תפריט עם שתי אפשרויות: תרגול / למידה) |
| `src/components/TopicCard.tsx` | הוספת כפתור למידה קטן בכרטיס הנושא |

### מבנה דף TopicLearn:
- עובר על כל השאלות של הנושא
- לכל שאלה מציג **רק** את כרטיס התיאוריה + שאלות חימום
- מדלג על השאלה המלאה — עובר ישירות לשאלה הבאה
- בסוף מציג סיכום: "למדת X קונספטים! מוכן לתרגל?"
- כפתור שמנווט ל-`/topic/:topicId` (תרגול רגיל)

## 3. שיפורי הוראה

| קובץ | שינוי |
|------|-------|
| `src/components/TheoryCard.tsx` | הוספת דוגמה אינטראקטיבית קטנה — קוד Python קצר עם הפלט שלו, כדי שהתלמיד יראה את הקונספט בפעולה |
| `src/lib/theoryContent.ts` | הוספת שדה `example` (קוד + פלט) לכל נושא, כדי שהתיאוריה תכלול דוגמה חיה |
| `src/components/WarmupView.tsx` | הוספת הסבר קצר **לפני** שאלת החימום (מה אנחנו בודקים כאן?) |
| `src/pages/TopicPractice.tsx` | הוספת step indicator ויזואלי (תיאוריה → חימום → שאלה) בראש הדף כדי שהתלמיד ידע איפה הוא |

### Step Indicator:
```text
[📖 תיאוריה] ——→ [💡 חימום] ——→ [📝 שאלה]
   ●                ○                ○
```
- מדגיש את השלב הנוכחי
- נותן תחושת התקדמות ומבנה

### דוגמה אינטראקטיבית בתיאוריה:
- בלוק קוד Python קצר (2-3 שורות) עם "מה יודפס?" וחשיפת התשובה בלחיצה
- מאפשר לתלמיד לבדוק הבנה לפני שמתחיל חימום
