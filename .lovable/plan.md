

# הוספת קטגוריית "מושגים" נפרדת עם סדר למידה הגיוני

## מה נבנה
עמוד תרגול מושגים נפרד שמאפשר לתלמידים לתרגל רק שאלות תיאורטיות (concept questions), עם סדר השתלשלות הגיוני — מהבסיס לנושאים מורכבים יותר.

## סדר הלמידה המוצע
השאלות יוצגו בסדר הבא (מהקל למורכב):
1. **מעקב קוד** — מושגי בסיס: `print`, השמה, מעקב
2. **תנאים ולוגיקה** — `elif`, אופרטורים לוגיים, ביטוי טרנרי
3. **לולאות** — `for` vs `while`, `break`, `continue`
4. **רשימות** — tuple vs list, שיטות רשימה, slicing
5. **מתמטיקה** — אופרטורים מתמטיים (`%`, `//`, `**`)

התלמיד יעבור על כל המושגים בסדר הזה, עם אפשרות לראות התקדמות לפי קבוצות.

## שינויים טכניים

### 1. עמוד חדש: `src/pages/ConceptsPractice.tsx`
- מסנן רק שאלות עם id שמתחיל ב-`concept_`
- מסדר אותן לפי הסדר הנ"ל (tracing → conditions → loops → lists → math)
- מציג כותרת קבוצה כשעוברים לנושא חדש
- progress bar כללי + מונה שאלות
- בסיום — מסך סיכום

### 2. הוספה ל-Topics page (`src/pages/Topics.tsx`)
- כרטיס בולט בראש העמוד "📚 מושגי יסוד" עם תיאור "למד את המושגים הבסיסיים לפני שמתחילים לתרגל"
- מציג התקדמות (כמה concept questions נענו נכון)

### 3. הוספת route ב-`src/App.tsx`
- `/concepts` → `ConceptsPractice`

### 4. עדכון `src/data/questions.ts`
- הוספת helper: `getConceptQuestions()` שמחזיר רק שאלות concept מסודרות
