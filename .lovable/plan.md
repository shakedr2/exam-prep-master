

# הוספת שאלות מכוונות (Scaffolded Learning)

## הרעיון
לפני כל שאלה מלאה, המערכת תציג 2-3 שאלות חימום קצרות וקלות שבונות את הידע הנדרש בהדרגה. אחרי שהתלמיד עובר את שאלות ההכנה, הוא מגיע לשאלה העיקרית מוכן יותר.

## דוגמה לזרימה

```text
+----------------------------------+
|  שלב 1/3 - שאלת הכנה             |
|  "מה עושה range(len(lst))?"      |
|  [תשובה א] [תשובה ב] ...        |
+----------------------------------+
           |
           v
+----------------------------------+
|  שלב 2/3 - שאלת הכנה             |
|  "מה ערך lst[-1] עבור [3,1,5]?"  |
|  [תשובה]                         |
+----------------------------------+
           |
           v
+----------------------------------+
|  שלב 3/3 - השאלה המלאה!          |
|  mystery([1,2,3,4,5]) = ?        |
|  [קוד + תשובה]                   |
+----------------------------------+
```

## שינויים טכניים

### 1. עדכון מבנה הנתונים (`src/data/questions.ts`)

הוספת שדה `warmupQuestions` לכל סוג שאלה - מערך של שאלות quiz קצרות שמכינות לשאלה העיקרית:

```typescript
interface WarmupQuestion {
  question: string;
  code?: string;
  options: string[];
  correctIndex: number;
  explanation: string;
}

// הוספה לכל סוג שאלה:
export interface TracingQuestion {
  // ... שדות קיימים
  warmupQuestions?: WarmupQuestion[];
}
```

כל שאלה (בעיקר medium ו-hard) תקבל 2-3 שאלות חימום רלוונטיות. דוגמאות:

- **לפני שאלת mystery על היפוך רשימה:** "מה עושה lst[0], lst[-1]?" -> "מה עושה swap?" -> השאלה המלאה
- **לפני שאלת Pell:** "מה P(0) ו-P(1)?" -> "חשב P(2) = 2*P(1)+P(0)" -> כתוב את הפונקציה
- **לפני שאלת is_prime:** "מה זה מספר ראשוני?" -> "האם 6 ראשוני? למה?" -> כתוב/עקוב אחרי הקוד

### 2. עדכון זרימת התרגול (`src/pages/TopicPractice.tsx`)

הוספת שלב ביניים - `WarmupPhase`:

- כשנכנסים לשאלה, אם יש לה `warmupQuestions`, מוצגות קודם שאלות ההכנה אחת אחרי השנייה
- כל שאלת חימום היא quiz פשוט עם 2-3 אפשרויות
- אחרי סיום כל שאלות החימום, מוצגת השאלה המלאה עם הודעה "עכשיו אתה מוכן!"
- אם אין `warmupQuestions`, השאלה מוצגת ישירות כמו היום
- progress bar משני מראה את ההתקדמות בתוך שאלות ההכנה (שלב 1/3, 2/3, 3/3)

### 3. קומפוננטת WarmupView חדשה

קומפוננטה חדשה שמציגה שאלת חימום בעיצוב מובחן:
- רקע בצבע שונה (כחול בהיר) כדי להבדיל מהשאלה העיקרית
- תגית "שאלת הכנה" במקום "רב-ברירה"
- אנימציה חלקה בין שאלות החימום

### 4. כמות שאלות חימום לפי רמת קושי

| רמת קושי | מספר שאלות חימום |
|-----------|------------------|
| קל (easy) | 0-1 שאלות |
| בינוני (medium) | 1-2 שאלות |
| קשה (hard) | 2-3 שאלות |

### 5. הוספת שאלות חימום לכל השאלות הקיימות

אעבור על כל ~55 השאלות הקיימות ואוסיף שאלות חימום מתאימות. כל שאלת חימום תתמקד במושג אחד קטן שנדרש לפתרון השאלה העיקרית, כגון:
- הבנת אופרטורים (`%`, `//`, `**`)
- הבנת פונקציות Python (`range`, `len`, `str`)
- הבנת מבני נתונים (slicing, indexing)
- הבנת לוגיקה (תנאים, לולאות)
